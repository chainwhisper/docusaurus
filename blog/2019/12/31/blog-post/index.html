Binance Chain Community

Binance Chain 2019 Annual Report

Check out highlights from Binance Chain’s 2019

Binance Chain is a blockchain project initiated by Binance and has since involved a whole community of Binancians. Binance Chain is designed as an alternative marketplace for issuing, using, and exchanging digital assets in a decentralized manner. The first use case for Binance Chain is Binance DEX, a new decentralized exchange (DEX) that aims to offer the benefits and security of a DEX and also carry over the best features of an exchange in a decentralized setting. Since launch, Binance DEX continues to build momentum as the leading global decentralized exchange with your support.

2019 was a year of blockchain technology exploration, increased use cases, and growing in research space. This post takes you on a year-long journey of Binance Chain’s 2019

February

  • Binance Launches DEX Testnet for the new era of peer-to-peer cryptocurrency trading February 20
  • Release of developer-kit:
    • A new repository on GitHub to host binaries
    • Binance Chain Testnet Explorer: https://testnet-explorer.binance.org
    • Documentation site: https://docs.binance.org/index.html
    • A bunch of tutorials is published: https://www.binance.vision/search/binance%20dex

March

  • Binance DEX Simulated Trading Competition https://www.binance.com/en/support/articles/360024724031
  • Binance Chain Coding Competition:
    • SDK solutions available in popular programming languages: Golang, Java, JavaScript, C++
  • Binance chain Security Bug Bounty Program is launched: https://www.binance.com/en/support/articles/360024789131

April

  • Binance Chain Mainnet Launched on April 18th: https://community.binance.org/topic/44/binance-chain-mainnet-swap
  • Binance Chain Mainnet Swap has begun
  • Binance Chain Evolution Proposal 2 (BEP2) protocol is published
  • Token listing guide has been published on the forum to explain the details of the essential steps.
  • First trading pair on Binance DEX: https://www.binance.org/en/trade/MITH-C76_BNB
  • Wallet support:
    • Hardware Wallets: Ledger, Cool Wallet
    • Software Wallets with DEX Support: Coinomi, Infinito, ZelCore
    • Others: MathWallet, Ellipal, Atomic, Exodus, HooWallet

May

  • Release v0.5.9 of binaries of Binance chain node, download here
  • A deep dive on match engine logic of Binance DEX is also published on Binance academy
  • Binance Chain Evolution Proposal (BEP) is an effective way for users to know the current status of a given feature, function or improvement. There are several PRs on GitHub repo. Binance Chain welcomes more contributions from the community.
  • ZenGo research team added BEP2 BNB token support to their threshold Signature (TSS). GitHub link
  • Fantom team have released an open source BEP2 migration bridge
  • Wallet Supports
    • Hardware Wallets: Ledger Nano X
    • Software Wallets with DEX Support: DDEX Wallet MEET.ONE Wallet
    • Others: Guarda Wallet

June

  • Announcement of reduction of fees to lower the barrier to entry and encourages= new project proposals
  • Binance issued pegged BTCB on Binance Chain. This will start with a Bitcoin-pegged token that will be traded on Binance and proposed for trading on Binance DEX.
  • USDSB is the first stablecoin issued by Stably on Binance Chain
  • Release v0.6.0 of binaries of Binance chain node, download here https://github.com/binance-chain/node-binary What’s new:
    • Matching Engine Revision: As explained in BEP19, Taker and Maker concepts have been introduced in the matching logic. The new match logic will be more friendly to new users and reduce surprises.
    • Delist Trading Pairs on Binance DEX: As explained in BEP6, if a token has a credit issue or one of its trading pairs has little trading volume for a long time, the community might consider dropping this trading pairs to improve the overall liquidity.
    • Time Locking of Token Assets on Binance Chain: As explained in BEP9, some projects decide to lock certain amount tokens for pre-defined periods of time, with the vesting in the future according to a pre-determined schedule. Documentation link
  • State Sync Enhancement: As explained in BEP18, the existing state sync implementation is refactored to improve user experience.
  • Release of airdop tool and token management GUI ,you can download from here
  • Binance.com Launches BEP2 Community Listing Program： To support the development of high-quality blockchain projects in the industry, Binance.com is beginning a new initiative to list projects on Binance.com, selected from the projects currently trading on Binance DEX.
  • Wallet supports
    • Mobile Wallets: Pundi X Wallet Arax Wallet
    • Chrome Extension wallet: Equal Wallet

July

  • Mainnet “Galileo” Upgrade to v0.6.0 Complete on 2019/7/15 ：

  • Release v0.6.1 of binaries of Binance chain node, download here https://github.com/binance-chain/node-binary What’s new? This version includes BEP12, which enables exchanges to reject deposits transactions that have no valid memo. If you are new to Binance Chain, the improved state-sync workflow made the syncing process more stable. Binance Chain command-line tool is more powerful now, and you can use it to generate a signed transaction and then use the hex string for broadcasting. Don’t forget to add the latest metrics for Tendermint on your monitoring tool.

  • Five tokenized currencies are now built on Binance Chain: TrueUSD, TrueGBP, TrueAUD, TrueCAD, and TrueHKD by TrustToken in July.

  • Loom’s PlasmaChain has recently integrated with Binance Chain, you can take a look at Loom SDK

  • Wallet developments

    • TrustWallet supports token swap
    • Mobile Wallets: TokenPocket
    • Hardware wallet: Safepal
    • Keyless: Zengo

August

  • Mainnet “Hubble” upgrade to v0.6.1 complete on 2019/08/05:
  • Introduce RFC 7692 compression algorithm to WebSocket server for Binance Chain.
  • Winner of BEP2 community listing program: Like ANKR, Your Token Can ‘DEX’ Its Way to Binance
  • Ethereum to Binance Chain bridge demo: https://forum.poa.network/t/ethereum-to-binance-chain-bridge-demo/2943
  • Binance DEX projcts analysis reports from TokenGazer and TokenInsight
  • Wallet supports
    • TrustWallet adds “auto-sign” feature
    • Mobile Wallets: Cobo Wallet, X Wallet

September

  • Release v0.6.2 of binaries of Binance chain node: download here What’s new? BEP3 was implemented and it defines native transactions to support Hash Timelock Contract (HTLC) on Binance Chain. As described in BEP39, this new feature is designed to add memo in the transfer data of WebSocket. If you are new to Binance Chain, the speed of the syncing process will be faster with hot-sync protocol. Don’t forget to add the latest metrics for Tendermint on your monitoring tool.
  • BUSD and pegged XRP is recently issued on Binance Chain
  • The first DAO on Binance Chain: https://medium.com/canyacoin/the-canyadao-is-now-live-and-ready-for-staking-9d5d446f26dc
  • Loom Network Is Bringing DeFi to Tron, Binance Chains : https://medium.com/loom-network/making-a-multichain-dai-maker-and-loom-network-to-build-dai-gateways-to-tron-binance-and-other-39ca20bd2a1d
  • Wallet supports
    • Trust wallet: Add Orderbook Support
    • Mobile Wallets: Midas Wallet, Cosmostation Wallet

October

  • Mainnet Archimedes Upgrade to v0.6.2 complete on 2019/10/14
  • Threshold Signature Scheme(TSS) on Binance Chain: Binance continues to contribute to open-source blockchain development, improving the security of funds and information for Binance Chain, Bitcoin networks and more. TSS library will enable wallet providers and custodians to avoid single points of failure in private keys with distributed key management. The full User Guide can be found here
  • CipherTrace Announces AML On-Chain Analytics for Binance Chain: CipherTrace, the world’s leading provider of cryptocurrency intelligence and blockchain security, today announced support for Binance Chain and its community that facilitates the transfer and trading of blockchain assets and BNB.
  • HTC Will Support Binance Chain With a Special Edition Smartphone: HTC’s EXODUS 1 - Binance Edition is the world’s first Binance Chain-ready smartphone for cryptocurrency enthusiasts and traders
  • Wallet supports:
    • Hardware Wallet: Trezor

  • Release v0.6.3 of binaries of Binance chain node: download here What’s new?This new version introduced substantial performance improvements, especially for a larger number of addresses and there is also a change of constraints about listing transactions is introduced. Binance DEX lot size will be improved to increase liquidity. The latest Release of Binance Chain Client has TSS Support. Please also pay attention to the latest changes in Tendermint RPC endpoint.
  • Mainnet Heisenberg Upgrade to v0.6.3 complete on 2019/11/27
  • Binance is collaborating with Bugcrowd for a new round of bug bounty to ensure a more robust security system for DEX trading platform.
  • Scalable and decentralized oracles for Binance API and Binance Chain using Band Protocol
  • Blockchain-powered smartphone with BNB supported:
  • Wallet supports:
